# Ressources de conformité (Artifacts)

Cette page regroupe les artefacts normatifs générés par l'IG :

- Profils FHIR : contraintes sur les ressources de base
- Extensions : éléments de données personnalisés
- Terminologies : ValueSets et CodeSystems utilisés
- Exemples : instances illustrant l'usage des profils

## Table des artefacts

La liste complète des artefacts générés est disponible dans le menu latéral ou via la page "Téléchargements".

Pour chaque artefact, vous trouverez :
- Une description fonctionnelle
- Les contraintes ou extensions appliquées
- Des exemples d'utilisation

Pour plus de détails sur la structure des artefacts, voir la page [Structure d'un IG](structure-ig.html).
